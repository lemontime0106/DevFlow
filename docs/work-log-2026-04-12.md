### 4. Google Analytics 연결 상태

현재 사이트에는 GTM 컨테이너만 설치된 상태다.

아직 완료되지 않은 부분:

- GA4 측정 ID 확인
- GTM 컨테이너 내부에 GA4 Google 태그 추가
- `All Pages` 트리거 연결
- GTM 변경사항 제출/게시
- GA4 실시간 보고서 또는 Tag Assistant로 수집 확인

메모:

- GA4 측정 ID는 `G-XXXXXXXXXX` 형식이다.
- `GTM-5X65KN66`은 Tag Manager 컨테이너 ID이며 GA4 측정 ID가 아니다.
- Firebase 프로젝트는 웹사이트 GA4 연결에 필수는 아니다. Firebase는 모바일 앱 분석이나 Firebase 기능 연동이 필요할 때 사용하면 된다.

### 5. 카테고리 관리 최소 버전

- `/settings`에서 사용자 커스텀 카테고리를 생성할 수 있게 서버 액션 추가
- 카테고리 이름은 1~40자로 정규화하고, 색상은 hex color만 허용
- 같은 사용자 기준 중복 이름이면 사용자용 에러 메시지를 반환하도록 처리
- settings 페이지에서 기본 카테고리와 사용자 카테고리 목록을 함께 표시
- 새 카테고리 저장 후 `/settings`, `/timer`, `/history`, `/reports/weekly`를 재검증하도록 연결
- 사용자 커스텀 카테고리 이름/색상 수정 액션 추가
- 기본 카테고리는 읽기 전용으로 표시
- 카테고리 생성/수정 실패 메시지를 토스트로 표시

### 6. 대시보드 목표 표시 개선

- 오늘 목표가 있을 때 남은 집중 시간을 표시
- 목표를 달성한 경우 달성 문구 표시
- 목표가 없을 때 `목표 -분`처럼 보이던 문구를 사용자용 안내로 교체
