# DevFlow 남은 작업 큐

작성일: 2026-04-12

## 목적

이 문서는 지금 코드 상태에서 추가로 진행할 작업을 실행 순서 중심으로 정리한다.
넓은 백로그는 `docs/next-work-roadmap-2026-04-12.md`를 기준으로 보고, 실제 다음 작업 선택은 이 문서를 기준으로 한다.

## 현재 기준 완료된 것

- 회원가입, 로그인, 로그아웃 흐름
- 이메일 미인증 안내와 인증 메일 재전송
- 세션 시작, 일시정지, 재개, 취소, 완료 저장
- 히스토리 조회와 필터링
- 일간 대시보드 집계
- 주간 리포트 집계와 기본 인사이트
- Settings에서 일일 목표 저장
- Settings에서 사용자 커스텀 카테고리 생성
- Settings에서 사용자 커스텀 카테고리 이름/색상 수정
- Settings에서 기본/사용자 카테고리 목록 표시
- 대시보드에서 오늘 목표까지 남은 집중 시간 표시
- GTM 컨테이너 설치와 Production 배포

## 메모

- GA4는 측정 ID가 확보될 때까지 대기한다.
- DB 스키마가 필요한 작업은 SQL 문서와 Supabase 타입 갱신을 같이 처리한다.
- 다음 개발 우선순위는 카테고리 삭제/비활성화 정책 결정과 Settings preset 확장이다.
